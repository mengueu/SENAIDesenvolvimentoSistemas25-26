Quadrado Camaleão

  • Passar o mouse em cima muda a cor aleatóriamente
  • Tirar o mouse muda a cor aleatóriamente
  • Clique no quadrado para aumentar seu tamanho
  • Clique com o botão direito para diminuir seu tamanho
  • Use as setas do teclado para mover o quadrado
  • Pressione 'T' para rotacionar o quadrado
  • Pressione 'C' para mudar a cor aleatóriamente
  • Pressione 'R' para reiniciar o quadrado
  • Mude o tamanho da janela para ver a largura atual
